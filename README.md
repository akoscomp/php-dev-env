# php-dev-env

## RUN:
* `git clone git@github.com:akoscomp/php-dev-env.git`
* `cd php-dev-env`
* `make build`

## Usage
Open in browser for adminer, php76 and php80 site:
* http://localhost:8000/
* http://localhost:8076/
* http://localhost:8080/

## Customize:
Add your php files in ./data/webroot/php74 and ./data/webroot/php80 folders.
